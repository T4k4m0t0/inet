Detectar red

1.- En el shell ejecuta el comando iwconfig y fijate que tu tarjeta wireless este en modo "managed".

2.- Tras esto, ejecuta el comando wlassistant que te escanea las redes y es bastante cómodo de utilizar.



Backtrack | Detectar red Intel 3945ABG

BackTrack detecta la tarjeta wifi, puedes verlo haciendo un lspci | grep -i 3945ABG
lo que tienes que hacer para que empieze a funcionar es ir a
Start > Backtrack > Radio Network Analysis > 80211 > Misc > Load IPW3945

Después abre el kwifimanager y verás que ya está activada :P
