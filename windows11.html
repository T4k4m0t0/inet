How to disable Recall in Windows 11 (Registry script to turn off Recall AI)

Group Policy Editor
Computer Configuration > Administrative Templates > Windows Components > Windows AI
Allow Recall to be enabled [X] Disabled
Restart PC
Usando el Registro
  1. Navegar hasta el path:
    2. Equipo\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsAI
  2. Haga clic derecho en el panel lateral derecho y seleccione Nuevo > DWORD (32-bit) Value y nombrelo como AllowRecallEnablement
  3. Haga doble clic en el valor recién creado y configúrelo.Value Data a 0. Mantener la base como Hexadecimal
  4. Cierre el editor de registro y restart PC para plicar cambios.

Disable-WindowsOptionalFeature -Online -FeatureName "Recall" -Remove
