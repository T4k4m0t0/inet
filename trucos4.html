Cambiar MTU ( Maximum Transmission Unit )
CMD permisos administrativos
netsh interface ipv4 show subinterfaces

o

netsh interface ipv4 show interface

esto nos da los valores e interfaces, para cambiar los valores MTU:

netsh interface ipv4 set subinterface "Local Area Connection" mtu=nnnn store=persistent

Donde:
Local Area Connection es el nombre de la conexion de tu computador, nnnn es el valor MTU

Borrar el fichero de hibernacion (hiberfil.sys)
Este TIP es útil para los sistemas de escritorio que nunca hibernan.

El fichero de hibernación (hiberfil.sys) tiene el mismo tamaño que la memoria física, por tanto,
si tenemos 2 GB de memoria, tendremos en la carpeta raíz de nuestro disco duro del sistema
un archivo llamado hiberfil.sys (oculto y del sistema) de ese tamaño.

Desactivar.:

o
powercfg -h off
