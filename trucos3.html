Activar actualizaciones automáticas

  • Inicio → Panel de control → Rendimiento y mantenimiento → Sistema → Actualizaciones automáticas → "Descargar las actualizaciones automáticamente y notificarme cuando estén listas para ser instaladas".

Activar ClearType

  • Inicio → Panel de Control → Propiedades de pantalla → Apariencia → Efectos
  • Activar "Use el siguiente método para suavizar los bordes de las fuentes de pantalla" → Seleccionar ClearType

Activar DMA en los discos

  • Inicio → Mi PC → Botón derecho → Administrar → Administrador de dispositivos → Controladores IDE/ATAPI → Canal IDE primario / secundario → Botón derecho → Propiedades → Configuración avanzada → DMA si disponible

Activar o desactivar la hibernación

  • Inicio → Panel de Control → Rendimiento y Mantenimiento → Opciones de Energía → Hibernación
  • NOTAS:
    • Esta función crea el fichero C:\hyberfile.sys de un tamaño igual a la RAM que tenga el equipo
    • Al darle a Inicio → Apagar, hay que presionar Mayúsculas para que "Suspender" se convierta en "Hibernar"
 

Activar o desactivar la Pantalla de bienvenida

  • Inicio → Panel de control → Cuentas de usuario → Cambiar la forma en que los usuarios se validan en el sistema

Bajar más ficheros a la vez de servidores HTTP

  • Inicio → Ejecutar → Regedit → HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings
  • Edición → Nuevo → Valor DWORD → MaxConnectionsPerServer → Doble click → 4 o más
  • Edición → Nuevo → Valor DWORD → MaxConnectionsPer1_0Server → Doble click → 4 o más

Cambiar el puerto que usa Escritorio remoto (por defecto es el 3389)

  • Inicio → Ejecutar → Regedit → HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\TerminalServer\WinStations\RDP-Tcp\PortNumber

Cambiar la letra de una unidad

  • Mi PC → Botón derecho → Administrar → Administrador de discos → Botón derecho → Cambiar letra de unidad

Cambiar la ubicación del Escritorio, Favoritos, Mis Documentos, Mis Imágenes, etc.

  • Inicio → Ejecutar → Regedit → HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders

Cambiar nombre de usuario y carpeta asociada

  • Mi PC → Botón derecho → Administrar → Usuarios Locales y Grupos → Usuarios → F2

Configurar la memoria virtual

  • Inicio → Panel de control → Rendimiento y mantenimiento → Sistema → Avanzado → Rendimiento → Avanzado → Memoria virtual

Convertir unidad de FAT32 a NTFS

  • Inicio → Ejecutar → CONVERT X: /FS:NTFS
  • X: es la unidad que queremos convertir, necesita reiniciar para hacerlo

Desactivar carpetas compartidas administrativas (C$, D$, etc)

  • Inicio → Ejecutar → Regedit
  • HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters
  • Edición → Nuevo → Valor DWORD → AutoShareWks → 0

Desactivar envío de informe de errores a Microsoft

  • Inicio → Mi PC → Propiedades → Avanzadas → Informe de errores → Desactivar envío de informe de errores

Desactivar mensajes de notificación

  • Inicio → Ejecutar → Regedit → HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced
  • Edición → Nuevo → Valor DWORD → EnableBalloonTips → 0

Desactivar la petición de usuario y contraseña (autologon)

  • Inicio → Ejecutar → control userpasswords2
  • Desmarcar  "Los usuarios deben introducir un nombre y una contraseña para acceder a este equipo" → Aplicar
  • En la ventana que aparece, poner el usuario que se quiera y su contraseña

Deshabilitar el servicio mensajero (para evitar mensajes de spam)

  • Mi PC → Botón derecho → Administrar → Servicios y aplicaciones → Servicios → Mensajero → Doble click → Tipo de inicio → Deshabilitado

Desinstalar Windows Messenger

  • Inicio → Ejecutar → RunDll32 advpack.dll,LaunchINFSection %windir%\INF\msmsgs.inf,BLC.Remove

Ejecutar programa minimizado o maximizado

  • Nuevo → Acceso directo → start programa /min
  • Nuevo → Acceso directo → start programa /max

Fijar tamaño de ventanas nuevas

  • Abrir ventana nueva → Cerrar las demás → Cambiar el tamaño de esta → Cerrar con Control+Click en la X

Habilitar cookies

  • Internet Explorer → Herramientas → Opciones de Internet → Privacidad → Nivel → Medio / Medio alto

Instalar Internet Information Services (IIS)

  • Inicio → Panel de Control → Agregar o quitar programas → Agregar o quitar componentes de Windows → Internet Information Services

Mantener el código del kernel de Windows siempre en la RAM

  • Inicio → Ejecutar → Regedit → HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management
  • DisablePagingExecutive → Doble click → 1

Mostrar la pestaña "Seguridad"

  • Inicio → Panel de control → Opciones de carpeta → Ver → Utilizar uso compartido simple de archivos [recomendado] → Desmarcar

Poner una secuencia de imágenes como protector de pantalla

  • Botón derecho en el escritorio → Propiedades → Protector de pantalla → Mis imágenes
  • Configuración → Elegir frecuencia de cambio, tamaño y efectos de transición

Poner un mensaje al inicio del sistema

  • Inicio → Ejecutar → regedit
  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon
  • Modificar los valores LegalNoticeCaption y LegalNoticeText,

Ejecuta System Restore

  • %SystemRoot%\System32\restore\rstrui

Aumento rendimiento red

  • HKEY_LOCAL_MACHINE\System\CurrentControl\St/Services//LanmanWorkstation/Parameters
    crear o modificar valor Dword MaxCmd y MaxThreads con numero decimal 100 (entre 0 y 255) por defecto 15

Mejorar cache L2

  • HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Session Manager\Memory Management
    crear o modificar valor Dword SecondLevelDataCache
    valor equivalente en Kbytes valor cache L2 procesador

Quitar el número de mensajes no leidos de la pantalla de bienvenida

  • Inicio → Ejecutar → regedit
  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\UnreadMail (todos los usuarios)
  • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\UnreadMail (mi usuario)
  • Crear un nuevo valor de nombre MessageExpiryDays, tipo DWORD y valor 0

Resetear la configuración del TCP/IP

  • netsh int ip reset log.txt

Utilizar modo de compatibilidad

  • Acceso directo al programa → Propiedades → Compatibilidad → Ejecutar este programa en modo compatibilidad

Ver dispositivos ocultos

  • Mi PC → Botón derecho → Propiedades → Hardware → Administrador de dispositivos → Ver → Mostrar dispositivos ocultos
